  • 'Monitor 5' may be used for diagnosis of Sinumerik 840D, Sinamics or Step7 systems connected via Ethernet.


    No installation is needed, the program is portable and saves all data in .INI files in own folder.
    Monitor5 is using Snap7 under LGPL3 Licence to communicate with the Sinumerik/Sinamics/Step7 equipment.
    Monitor5 is using Qt under LGPL3 Licence.

  • 'Cam410' is a package of very small simulators for the Step7/TIA trainings. One simulator is for the Step7 "training conveyor", another one for Safety Intergated "training production unit", etc.

    No installation is needed, each program is portable and saves all data in .INI files in own folder.
    'Cam410' is using Snap7 under LGPL3 Licence to communicate with the Sinumerik/Sinamics/Step7 equipment.
    'Cam410' is using Qt under LGPL3 Licence.
